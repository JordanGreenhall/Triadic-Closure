## Path rule

The former Google Drive folder paths are historical metadata. In this repository, resolve corpus documents by root-level filename unless GitHub visibly shows an actual subdirectory. For example:

```text
mass-as-self-closure.md
```

not:

```text
results/mass-as-self-closure.md
```

Do not add an index entry until the target file has been verified in the current repository tree.
